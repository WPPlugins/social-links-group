== Description ==

SOCIAL LINKS GROUP:

Social Links Group provides functionality to users to make groups of social icons for different socials profiles with customizable design. It helps when you have multiple social profile to display on your site.It is very easy to customize even for non-technical person.

You will be able to customize Icons Size, Icons shape, Icons Position, Icons Layout, Icons Direction and more with this plugin.

Live Demo: <a href="http://plugins.auratechmind.net/social-links-group/"> Click Here </a><br>

== Installation ==

Steps to Install Plugin.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. GO to Settings -> Social links
4. Add group, and add links and save it.
5. Place generated shortcode anywhere on page or post.
